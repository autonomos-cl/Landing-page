 Autonomos.dev - Soluciones de IA Empresarial

## Descripción
Sitio web corporativo para Autonomos.dev, una empresa especializada en implementar soluciones de IA generativa y automatización para empresas. El sitio presenta los servicios, portafolio y información de contacto de la compañía.

## Características Principales
- Diseño moderno y responsivo
- Navegación suave con scroll
- Portafolio interactivo con modales
- Formulario de contacto y suscripción
- Integración con redes sociales
- Chat bot integrado
- Optimizado para SEO

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript
- Botpress (Chat bot)
- Masonry.js (Grid del portafolio)
- Smooth Scroll

## Estructura del Sitio
- Header con navegación
- Sección Hero
- Sobre Nosotros
- Servicios
- Portafolio de Trabajos
- Formulario de Contacto
- Footer con información adicional
